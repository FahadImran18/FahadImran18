## Projects

**[Daira — Pakistan Real Estate Platform](https://github.com/FahadImran18/Daira)**  
Role-based property platform with real-time chat, AI-powered insights, 360° virtual tours, and Stripe payments. Built with Next.js, Supabase, Gemini AI, and Google Maps.

**[Stock Prediction RPS](https://github.com/FahadImran18/stock-prediction-rps)**
